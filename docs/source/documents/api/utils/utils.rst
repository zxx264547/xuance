Utils
======================

In this part, we provide some mathematical basic tools for DRL algorithms.

.. toctree::
    :hidden:

    Distributions <distributions>
    Input Reformation <input_reformat>
    Layers <layers>
    Operations <operations>
    Value Norm <value_norm>
    

* :doc:`Distributions <distributions>`: Contains the tools for distributions of stochastic variables.
* :doc:`Input Reformation <input_reformat>`: Get the input list for representations and policies.
* :doc:`Layers <layers>`: Create a block of MLP, CNN, or RNN layer.
* :doc:`Operations <operations>`: Contains some basic tools.
* :doc:`Value Norm <value_norm>`: Normalize the values for actor-critic based algorithms.
